## Directory Structure

- `src/` — Models, datasets, datamodules, plmodules, callbacks, utilities.
- `configs/` — Hydra configuration files.
- `docs/skills/` — Project guidelines and conventions (auto-discovered by Claude Code).
- `docs/reference/` — Reference materials from external projects.
- `scripts_core/` — Example running scripts.
- `scripts_nus/` — NUS cluster-specific scripts (internal use).
- `requirements/` — Dependencies and pre-commit config.
